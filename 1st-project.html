Overview

This project allows you to remotely control a door using ESP32-CAM and monitor the area via a security camera. It integrates a servo motor, FTDI adapter for programming, and an IR sensor to detect motion.

Components Needed

  • ESP32-CAM
  • FTDI Adapter
  • Servo Motor
  • IR Sensor
  • L298N Motor Driver
  • Buttons for Manual Control

Arduino IDE Setup

Follow these steps to set up and upload code to your ESP32-CAM using the FTDI adapter:

  1. Install ESP32 board library in Arduino IDE.
  2. Select the correct board (ESP32-CAM) and the right COM port.
  3. Connect the FTDI adapter to ESP32-CAM, wiring RX to TX and TX to RX.
  4. Upload the code (sample code shown below).
  5. Remove the jumper wires after uploading and restart ESP32-CAM to run the code.

Programming ESP32-CAM with FTDI

Ensure correct wiring: Connect RX of FTDI to TX of ESP32-CAM and TX of FTDI to RX of ESP32-CAM. Power the ESP32 with 5V and GND pins. Place the jumper wire for flashing and remove it after the code is uploaded.

Where to Use This Project

This project can be applied in home automation, office security, and anywhere remote monitoring and door control are required.

Advantages

  • Remote door access and monitoring.
  • Enhanced security with automatic IR sensor detection.

Disadvantages

  • Dependent on Wi-Fi for remote access.
  • Initial setup complexity.
